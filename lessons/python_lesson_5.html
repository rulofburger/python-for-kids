8  Lesson 5: Lists and For Loops

9 Learning goals

By the end of this lesson you should be able to:

  • create a list of values
  • use a for loop to repeat code for each item in a list
  • understand the difference between while loops and for loops
  • use range() to repeat code a fixed number of times
  • build small programs using lists of names, scores, and items

Estimated time: 45–60 minutes

10 Part 1: Review from Lesson 4

In Lesson 4, you learned how to repeat code using a while loop.

count = 1

while count <= 5:
    print(count)
    count = count + 1

A while loop is useful when we want to repeat code while a condition remains true.

Today we learn another kind of loop: the for loop.

11 Part 2: What is a list?

A list stores several values in one variable.

names = ["Alex", "Ben", "Chloe"]

The variable names contains three pieces of text.

You can print the whole list:

names = ["Alex", "Ben", "Chloe"]

print(names)

Output:

['Alex', 'Ben', 'Chloe']

12 Part 3: A first for loop

A for loop lets us do something with each item in a list.

names = ["Alex", "Ben", "Chloe"]

for name in names:
    print("Hello,", name)

Output:

Hello, Alex
Hello, Ben
Hello, Chloe

Read this as:

For each name in the list called names, print a greeting.

13 Part 4: Lists of numbers

Lists can also contain numbers.

scores = [8, 10, 7, 9]

for score in scores:
    print(score)

You can do calculations inside the loop.

scores = [8, 10, 7, 9]

for score in scores:
    print("Double score:", score * 2)

14 Part 5: Adding up numbers in a list

A common pattern is to start with a total of 0 and then add each value.

scores = [8, 10, 7, 9]

total = 0

for score in scores:
    total = total + score

print("Total:", total)

Output:

Total: 34

15 Part 6: Using range()

Sometimes we want to repeat code a fixed number of times.

for count in range(5):
    print("Python")

This prints "Python" five times.

Important: range(5) gives the numbers 0, 1, 2, 3, and 4.

for number in range(5):
    print(number)

Output:

0
1
2
3
4

If you want to count from 1 to 5, use:

for number in range(1, 6):
    print(number)

Output:

1
2
3
4
5

16 Part 7: while loop or for loop?

Use a while loop when you do not know in advance how many times the loop must run.

Example: keep asking for a password until it is correct.

password = ""

while password != "letmein":
    password = input("Enter the password: ")

Use a for loop when you want to go through a list, or repeat something a known number of times.

Example: greet every name in a list.

names = ["Alex", "Ben", "Chloe"]

for name in names:
    print("Hello,", name)

17 Part 8: Common mistakes

17.1 Mistake 1: Forgetting the colon

Wrong:

for name in names
    print(name)

Correct:

for name in names:
    print(name)

17.2 Mistake 2: Forgetting indentation

Wrong:

for name in names:
print(name)

Correct:

for name in names:
    print(name)

17.3 Mistake 3: Expecting range(5) to include 5

for number in range(5):
    print(number)

This prints 0 to 4, not 1 to 5.

Use:

for number in range(1, 6):
    print(number)

if you want 1 to 5.
