6  Lesson 4: Repeating Code with Loops

7 Learning goals

By the end of this lesson you should be able to:

  • understand why loops are useful
  • use a while loop to repeat code while something is true
  • update variables inside a loop
  • avoid infinite loops
  • use loops to build simple games and repeated quizzes

Estimated time: 45–60 minutes

8 Part 1: Review from Lesson 3

In Lesson 3, you learned how to make Python choose between different options.

age = int(input("How old are you? "))

if age >= 13:
    print("You can join the coding club.")
else:
    print("You are not old enough yet.")

Today we make Python repeat actions.

This is one of the most powerful ideas in programming.

9 Part 2: Why loops are useful

Suppose you want to print this:

Python is fun!
Python is fun!
Python is fun!
Python is fun!
Python is fun!

You could write:

print("Python is fun!")
print("Python is fun!")
print("Python is fun!")
print("Python is fun!")
print("Python is fun!")

But that is boring and repetitive.

A loop lets us write the instruction once and repeat it.

10 Part 3: A first while loop

A while loop repeats code while a condition is true.

count = 1

while count <= 5:
    print("Python is fun!")
    count = count + 1

Output:

Python is fun!
Python is fun!
Python is fun!
Python is fun!
Python is fun!

The variable count starts at 1.

Each time the loop runs, we add 1 to count.

Eventually count becomes 6, so count <= 5 is no longer true.

Then the loop stops.

11 Part 4: Printing the counter

It is often useful to print the counter while the loop runs.

count = 1

while count <= 5:
    print("Count is", count)
    count = count + 1

Output:

Count is 1
Count is 2
Count is 3
Count is 4
Count is 5

12 Part 5: Counting down

Loops can also count down.

count = 5

while count > 0:
    print(count)
    count = count - 1

print("Blast off!")

Output:

5
4
3
2
1
Blast off!

13 Part 6: Loops with user input

Loops are useful when we want to keep asking the user something.

password = ""

while password != "python123":
    password = input("Enter the password: ")

print("Access granted.")

The loop keeps running while the password is not equal to "python123".

When the user finally types the correct password, the loop stops.

14 Part 7: Common mistakes

14.1 Mistake 1: Forgetting to update the variable

Wrong:

count = 1

while count <= 5:
    print(count)

This loop never stops, because count always stays 1.

Correct:

count = 1

while count <= 5:
    print(count)
    count = count + 1

14.2 Mistake 2: Indenting too much

Wrong:

count = 1

while count <= 5:
    print(count)
    count = count + 1
    print("Done")

This prints "Done" every time the loop runs.

Correct:

count = 1

while count <= 5:
    print(count)
    count = count + 1

print("Done")

This prints "Done" once, after the loop has finished.

14.3 Mistake 3: Using the wrong condition

This loop does not run at all:

count = 1

while count > 5:
    print(count)
    count = count + 1

Why? Because count > 5 is already false at the start.
