4  Lesson 2: Making Python Interactive

5 Learning goals

By the end of this lesson you should be able to:

  • use input() to ask the user questions
  • store answers in variables
  • combine input with print()
  • convert text input into numbers using int()
  • build a simple interactive calculator

Estimated time: 45–60 minutes

6 Part 1: Review from Lesson 1

In Lesson 1, you learned that Python can print text and do calculations.

name = "Emma"
age = 15

print(name, "is", age, "years old.")

Output:

Emma is 15 years old.

But so far, the program only knows what you wrote into the code.

Today we make programs that ask the user questions.

7 Part 2: The input() function

The input() function asks the user to type something.

name = input("What is your name? ")

print("Hello,", name)

When you run it, Python asks:

What is your name?

The user types an answer.

Then Python stores the answer in the variable name.

8 Part 3: Input is always text

This is very important.

When Python uses input(), it stores the answer as text, even if the user types a number.

Try this:

age = input("How old are you? ")

print(age)

This works.

But this does not work properly:

age = input("How old are you? ")

next_year = age + 1

print(next_year)

Python gets confused because age is text, not a number.

9 Part 4: Converting input into a number

To turn text into a whole number, use int().

age = input("How old are you? ")

age = int(age)

next_year = age + 1

print("Next year you will be", next_year)

You can also write it more shortly:

age = int(input("How old are you? "))

print("Next year you will be", age + 1)

10 Part 5: A simple calculator

Now we can build interactive programs.

number1 = int(input("Enter the first number: "))
number2 = int(input("Enter the second number: "))

total = number1 + number2

print("The total is", total)

Try running it with different numbers.

11 Part 6: Common mistakes

11.1 Mistake 1: Forgetting quotation marks

Wrong:

name = input(What is your name?)

Correct:

name = input("What is your name? ")

11.2 Mistake 2: Forgetting int()

Wrong:

age = input("How old are you? ")
print(age + 1)

Correct:

age = int(input("How old are you? "))
print(age + 1)

11.3 Mistake 3: Typing words when Python expects a number

This will crash if the user types "twelve":

age = int(input("How old are you? "))

For now, type numbers using digits, like:

12
15
42
