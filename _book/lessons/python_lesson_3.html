6  Lesson 3: Making Decisions with if

7 Learning goals

By the end of this lesson you should be able to:

  • use if to make Python choose between options
  • use else for the alternative case
  • use comparison operators like >, <, and ==
  • use elif when there are more than two possibilities
  • build a simple quiz and decision-making game

Estimated time: 45–60 minutes

8 Part 1: Review from Lesson 2

In Lesson 2, you learned how to ask the user for information.

name = input("What is your name? ")

print("Hello,", name)

You also learned that input is always text, so numbers need int().

age = int(input("How old are you? "))

print("Next year you will be", age + 1)

Today we make programs that can make decisions.

9 Part 2: The if statement

An if statement lets Python decide whether to run some code.

age = int(input("How old are you? "))

if age >= 13:
    print("You are old enough to use this app.")

The line:

if age >= 13:

means:

If age is greater than or equal to 13, run the indented code below.

The spaces before print() are important. They show Python which code belongs inside the if.

10 Part 3: Comparison operators

Python can compare values.

Operator Meaning
> greater than
< less than
>= greater than or equal to
<= less than or equal to
== equal to
!= not equal to

Important: use == when checking whether two things are equal.

answer = input("What is the capital of France? ")

if answer == "Paris":
    print("Correct!")

Do not use = here.

= stores a value.

== compares two values.

11 Part 4: Using else

Sometimes we want one thing to happen if the condition is true, and another thing to happen if it is false.

age = int(input("How old are you? "))

if age >= 13:
    print("You are old enough.")
else:
    print("You are not old enough yet.")

The else means:

Otherwise, do this.

12 Part 5: Using elif

Sometimes there are more than two possibilities.

mark = int(input("Enter your test mark: "))

if mark >= 80:
    print("Excellent!")
elif mark >= 60:
    print("Good work.")
elif mark >= 50:
    print("You passed.")
else:
    print("You did not pass yet.")

Python checks the conditions from top to bottom.

As soon as one condition is true, Python runs that block and skips the rest.

13 Part 6: Common mistakes

13.1 Mistake 1: Forgetting the colon

Wrong:

if age >= 13
    print("Old enough")

Correct:

if age >= 13:
    print("Old enough")

13.2 Mistake 2: Forgetting indentation

Wrong:

if age >= 13:
print("Old enough")

Correct:

if age >= 13:
    print("Old enough")

13.3 Mistake 3: Using = instead of ==

Wrong:

if answer = "Paris":
    print("Correct")

Correct:

if answer == "Paris":
    print("Correct")

15 Challenges

15.1 Challenge 1: Mini quiz

Create a three-question quiz.

Ask three questions, one at a time.

Keep a score variable.

Add 1 to the score for each correct answer.

At the end, print the final score.

Example:

Your final score is 2 out of 3.

15.2 Challenge 2: Adventure choice game

Create a tiny adventure game.

The user starts at a fork in the road.

Ask whether they want to go left or right.

If they go left, something happens.

If they go right, something else happens.

Then ask one more question and give a final ending.

Example:

You are standing at a fork in the road.
Do you go left or right?
