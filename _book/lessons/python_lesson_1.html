4  Lesson 1: Your First Python Program

5 Learning goals

By the end of this lesson you should be able to:

  • write and run a Python program in Positron
  • use print() to display information
  • store information in variables
  • perform simple calculations
  • combine text and numbers to create useful output

Estimated time: 45–60 minutes

6 Part 1: Running your first program

Open Positron.

Create a new file called:

lesson1.py

Type:

print("Hello, world!")

Click Run.

You should see:

Hello, world!

Congratulations! You have written your first Python program.

7 Part 2: The print() function

A function is a command that tells Python to do something.

The print() function tells Python to display something on the screen.

Examples:

print("Python is fun!")
print(27)
print(3 + 4)

Notice the difference:

print("3 + 4")

prints:

3 + 4

while:

print(3 + 4)

prints:

7

The quotation marks tell Python: treat this as text.

Without quotation marks, Python treats it as mathematics.

8 Part 3: Variables

Imagine you have a labelled box.

Age
┌───────┐
│ 15    │
└───────┘

The label is the variable name.

The value inside is the value.

Example:

age = 15
print(age)

Output:

15

Variables can also store text.

name = "Emma"
print(name)

9 Part 4: Using variables

Variables make programs much easier to change.

Instead of:

print("Emma is 15 years old.")

we can write:

name = "Emma"
age = 15

print(name, "is", age, "years old.")

Output:

Emma is 15 years old.

Notice that print() automatically puts spaces between the different things you ask it to print.

10 Part 5: Doing calculations

Python is an excellent calculator.

print(8 + 4)
print(20 - 5)
print(6 * 7)
print(18 / 3)

You can also calculate using variables.

apples = 12
friends = 4

print(apples / friends)

Output:

3.0

11 Part 6: Updating variables

Variables can change.

score = 10

print(score)

score = score + 5

print(score)

Output:

10
15

The second assignment means: take the old score and add five.
